🚀 COMO FAZER DEPLOY NO NETLIFY
═══════════════════════════════════════════════════════════════

OPÇÃO 1: Deploy Automático via Git (Recomendado)
────────────────────────────────────────────────

Se o projeto já está conectado ao Git/GitHub:

  1. Abra o terminal na pasta do projeto

  2. Execute os comandos:
     cd /Users/juanminni/deploy-final-shieldcar/netlify-deploy
     git add .
     git commit -m "Update to v2.0 - Optimized layout"
     git push origin main

  3. Netlify fará deploy automático em 1-2 minutos!


OPÇÃO 2: Deploy Manual via Netlify CLI
───────────────────────────────────────

  1. Instale Netlify CLI (se ainda não tiver):
     npm install -g netlify-cli

  2. Faça login:
     netlify login

  3. Deploy:
     cd /Users/juanminni/deploy-final-shieldcar/netlify-deploy
     netlify deploy --prod


OPÇÃO 3: Deploy Drag & Drop
────────────────────────────

  1. Acesse: https://app.netlify.com

  2. Vá no site existente do ShieldCar

  3. Clique em "Deploys" → "Drag and drop"

  4. Arraste a pasta netlify-deploy
